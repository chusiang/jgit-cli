# jgit-cli

This is the **jgit-cli** [^1] deploy scripts for GNU/Linux and Mac OS X.

* [JGit/User Guide | Eclipsepedia](http://wiki.eclipse.org/JGit/User_Guide)

## Setup

1. Install.

        make install

2. Uninstall.

        make uninstall

[^1]: Java implementation of GIT version control (command-line).
